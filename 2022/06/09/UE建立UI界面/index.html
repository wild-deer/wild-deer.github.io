UE建立Ui界面

简单建立ui的过程

1.效果展示

2.实现步骤

​ 首先在content中新建一个文件夹用来存放ui界面的 widget，在这个文件夹中创建4个widget 命名 BackGrand、Button、Fade、Setting，然后在UI文件夹中建立一个子文件夹，命名为 Info ，在 Info 中新建两个widget命名为Infocontrol、InfoUI 。

BackGrand

​ 首先是菜单栏的背景制作。打开 BackGrand widget 后，在设计界面的控制面板中添加画布面板、背景模糊图片和文本界面，并按图中所示层级摆放，背景板的制作完成。

Button

​ 点开 Button widget 在设计界面控制面板中添加画布面板、垂直框、三组按钮和问题组件，并按如下层级摆放。锚点都正在屏幕左上角。

​ 添加四组 animation 动画分别控制三个菜单按钮在鼠标悬停时的动画显示和菜单的显示隐藏动画。

​ 首先设置动画 Hover1 ，点击 Hover1 ，点击设计界面中按钮，然后点击添加轨道，然后点击添加所属组件（文本），然后在新建立的轨道分别泰添加变换和颜色不透明度。

​ 添加好之后在时间为0处打上变换、颜色和不透明度的关键帧。将时间轴移动到 0.15 秒，并设置 x ，y 的缩放为 1.1，颜色改变为蓝色，然后打上关键帧。播放查看效果。

​ 可以看到动画效果正常，剩下的3个按钮的动画也是大同小异，可以直接CTRL+D复制动画，重命名为 hover…然后点击轨道上方的组件 TextBlock_0 然后选中设计界面中的需要替换的文本组件，右键点击之后选择替换为…即可。

​ 最后是 fade 动画，即隐藏该界面的动画，添加轨道，然后 K 画布面板的帧

Fade

​ 在 Fade 设计界面添加要 image 组件，添加动画命名为fade，设置图像的颜色为黑色，不透明，然后 k 图像的不透明度的帧，由0到1。显示动画效果。

setting

​ 复制 Button 组件，添加缺少的按钮和文本组件，并修改文字，然后返回

​ 在图表界面添加如下节点，Button_2为 画面质量极致按钮，点击之后设置画面质量机制，其中Get Game User Settings，为获得游戏设置对象，可以对游戏本地机进行设置，如分辨率、窗口模式、可延展性等，Set Overall Scalability Level为设置画面效果，其中 Value值越大效果越高，一般最高位3，最低为0，其余画面质量的设置大同小异。

​ 然后是返回按钮，下面是返回功能的整体蓝图。

​ 其中用到了 Set Visibility按钮，in visibaility 的值为非可命中测试仅自生，代表在按下返回按钮之后，禁用其余按钮。然后播放该界面消失的动画 fade ，在短暂延迟之后创建上级菜单界面，使用create Widget 组件，添加创建之前创建的 Button 按钮组件，附加到视口，然后逆向播放 Button 组件中的小时动画，在短暂延迟之后，将当前组件移除，使用 Remove from Parent组件。该组件消失后将不再占用资源。
